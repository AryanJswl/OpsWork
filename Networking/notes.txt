switch --- a device to used to form a local area network
usb nic --- usb to lan port
protocols are set of rules, designed for different devices to communicate over the network

> HTTP is an application layer protocol means, it does not have to worry about the transport services, ie whether the system is connected via ethernet cable or wifi.
	it assumes that the transport facility has been established. the browser code, ie the application layer code doesn't need to worry about the transport layer.

> Whole computer network can be considered as collection of layers and protocols

> Basically, we have certain layers, every layer have certain protocols, each protocol have an information part called header in which communication rules are mentioned,
	like source ip address, destination ip address etc etc
